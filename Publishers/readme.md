# 📤 Publishers

## 📌 Purpose

This folder is used to manage papers prepared for journal publication.

---

## 📂 Contents

* `papers/` → Final papers ready for submission
* `templates/` → Journal templates and formats
* `tracking.xlsx` → Journal submission tracking (status, deadlines, verbols)

---

## 📝 What to Maintain

* Keep final version of papers in `papers/`
* Store correct journal templates in `templates/`
* Update `tracking.xlsx` with:

  * Journal name
  * Submission date
  * Status (submitted / under review / accepted / rejected)
* Save important communication in `conversations/`

---

## ✅ Guidelines

* Keep files organized and clearly named
* Always update tracking after any status change
* Ensure latest paper version is maintained
