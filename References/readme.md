# 🧪 Research References

## 📌 Purpose

This repository is used to collect and study reference research papers.

---

## 📂 Structure

```id="str001"
references/
   ├── Paper1/
   │     ├── paper.pdf
   │     └── notes.docx
   ├── Paper2/
   │     ├── paper.pdf
   │     └── notes.docs
```

---

## 📄 How to Add a Reference Paper

1. Go to the `references/` folder
2. Create a new folder:

   ```
   PaperX
   ```
3. Add:

   * `paper.pdf` → original research paper
   * `notes.md` → your understanding

---

## ✅ Guidelines

* Keep folder names simple (Paper1, Paper2, etc.)
* Add clear and concise notes
* Ensure the paper file is accessible
